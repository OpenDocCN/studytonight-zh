## 下载

### Docker

```
docker pull apachecn0/studytonight-zh
docker run -tid -p <port>:80 apachecn0/studytonight-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g studytonight-zh
studytonight-zh <port>
# 访问 http://localhost:{port} 查看文档
```
