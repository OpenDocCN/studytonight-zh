# HTML Wbr 标记

> 原文：<https://www.studytonight.com/html5-references/html-wbr-tag>

在 HTML 中，`<wbr>`标记代表**断词机会。**断字机会基本上是文本中浏览器可以随意换行的位置。

*   `<wbr>`是 HTML 中的**新标签**。

*   需要注意的重要一点是`<wbr> tag` **没有在**行断点引入连字符**。**

*   这是一个空标签。

## HTML `<wbr>`标签语法和用法

因为它是一个空标签；因此它不需要结束标记。它只需要一个开始标记。因此，其语法如下:

```
<wbr>   content here
```
