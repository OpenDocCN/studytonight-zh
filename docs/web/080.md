# HTML `<p>`标签

> 原文：<https://www.studytonight.com/html5-references/html-p-tag>

在 HTML 中，`<p>`标签代表**段落**。

*   **段落**通常表示为一个**文本块**，通过使用空行或使用**首行缩进**将该文本块与文本的其余部分分开

*   另一方面，HTML 段落被表示为相关内容**的**结构分组**，如图像、表单**等。

*   `<p>`标签边距前后得到浏览器自动添加的**。**

*   `<p>`标签属于**块级元素**的范畴。

## HTML `<p>`标签语法和用法

`<p>`标签需要**开启(开始)标签**和**关闭(结束)标签。**

```html
<p>
...!content here
</p> 
```

## HTML

标签的默认 CSS 设置

`<p>`标签的默认 CSS 设置如下:

```html
p {
  display: block;
  margin-top: 1em;
  margin-bottom: 1em;
  margin-left: 0;
  margin-right: 0;
}
```
