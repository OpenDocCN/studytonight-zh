# C++ 程序：打印斐波那契数列

> 原文：<https://www.studytonight.com/cpp-programs/program-to-print-fibonacci-series-in-cpp>

以下是查找斐波那契数列的程序。

```cpp
#include<iostream.h>

int main()
{
    int n, t1=0, t2=1, nextTerm=0;

    cout << "Enter the number of terms: ";
    cin >> n;
    cout << "Fibonacci Series: ";

    for (int i=1; i <= n; ++i)
    {
        if(i == 1)
        {
            cout << " " << t1;
            continue;
        }
        if(i == 2)
        {
            cout << t2 << " ";
            continue;
        }
        nextTerm = t1 + t2;
        t1 = t2;
        t2 = nextTerm;

        cout << nextTerm << " ";
    }
    getch();
    return 0 ;
}
```

输入项数:10
斐波那契数列:0 1 2 3 5 8 13 21 34
