# 红宝石异常处理

> 原文：<https://www.studytonight.com/ruby/exception-handling-in-ruby>

**Ruby 中的异常处理**涉及到我们如何处理程序中出现的错误。通过错误，我们不是在谈论语法错误，例如拼写错误 **Ruby 关键字**。我们指的是程序无法恢复的错误，程序员不会预料到但它可能会发生。好的程序员预见到它们并采取行动来处理它们。

**异常**由于程序运行时系统无法处理的事情而发生。
