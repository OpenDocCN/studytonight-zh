# Ruby 中的迭代器

> 原文:[https://www.studytonight.com/ruby/iterator-in-ruby](https://www.studytonight.com/ruby/iterator-in-ruby)

`Iterators`允许我们遍历一个范围，并在一个范围内做一定次数的事情。演示`Iterator`比描述更容易。
